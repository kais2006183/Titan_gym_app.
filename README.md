# 🏋️‍♂️ Titan Gym App - نظام إدارة الصالات الرياضية الاحترافي

تطبيق متكامل وبنية برمجية ذكية مصممة خصيصاً لإدارة صالات بناء الأجسام واللياقة البدنية. تم بناء التطبيق كحزمة واحدة متكاملة باستخدام إطار عمل **Flutter** ليعمل بكفاءة عالية وبواجهة مستخدم عصرية ومظلمة (Dark Metallic Theme) تناسب أجواء الصالات الرياضية.

---

## ✨ ميزات التطبيق الرئيسية

* **📊 لوحة تحكم ذكية (Dashboard):** تعرض إحصائيات فورية شاملة لإجمالي المشتركين، الحالات الحرجة المنتهية، إجمالي الإيرادات، وصافي الأرباح.
* **👥 إدارة المشتركين والأبطال:** إمكانية إضافة مشتركين جدد، البحث الذكي برقم الهاتف أو الاسم، وتتبع تنازلي دقيق لدورة الاشتراك (30 يوماً).
* **💰 دفتر الحسابات والمالية الرئيسي:** نظام متكامل لتسجيل المداخيل والمصاريف يدوياً مع تصنيفات دقيقة (اشتراكات، مكملات، رواتب، فواتير، صيانة معدات).
* **🔔 مركز التنبيهات السريع:** شاشة مخصصة لعزل الحسابات الحرجة (المنتهية أو المتبقي لها أقل من 5 أيام) مع ميزة محاكاة إرسال رسائل تذكير مخصصة عبر الـ WhatsApp بضغطة زر.
* **🔐 حماية ودخول أوفلاين مدمج:** نظام تسجيل دخول محاكي يؤمن لوحة التحكم ويعمل بالكامل أوفلاين (Mock Data Environment) لضمان السرعة وسهولة التجربة.

---

## 🛠️ البنية التقنية والأدوات المستخدمة

* **Framework:** [Flutter](https://flutter.dev/) (Dart)
* **Design Pattern:** State Management محاكي مدمج لتسهيل الأداء (GymController).
* **Fonts:** [Google Fonts](https://pub.dev/packages/google_fonts) (خط Cairo للغة العربية، و خط Black Ops One للهوية البصرية).
* **Architecture:** ملف برمي موحد وموسع (تتجاوز بنيته المئات من الأسطر البرمجية الشاملة والمترابطة بدون أي اختصارات).
